A simple plugin designed to clear out empty custom post meta created during theme development.

== Description ==

A simple plugin which runs a sql query in order to clear out empty post meta that can be left over from the
theme development process.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
